## Usage

Stamp wraps all your package managers into a single CLI. Below are guides for each command.

- [Setup](/usage/setup.html) — First-time setup wizard
- [Configuration](/usage/configuration.html) — Config and manifest reference
- [Installing Packages](/usage/installing-packages.html) — Install with flags, notes, and aliases
- [Installing Repos](/usage/installing-repos.html) — Add third-party repositories
- [Removing Packages](/usage/removing-packages.html) — Remove and untrack
- [Removing Repos](/usage/removing-repos.html) — Remove repositories
- [Listing Packages](/usage/listing-packages.html) — View tracked packages
- [Reconcile](/usage/reconcile.html) — Track external installations
- [Restore](/usage/restore.html) — Rebuild on a new machine
- [Update](/usage/update.html) — Update all packages
- [Self-Update](/usage/self-update.html) — Update stamp itself
