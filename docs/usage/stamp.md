## stamp

A lightweight yet powerful wrapper for your native package managers

```
stamp [flags]
```

### Options

```
  -h, --help      help for stamp
  -j, --json      output results in JSON format
  -v, --verbose   enable debug logging
  -y, --yes       auto-accept all prompts
```
